## About MyLawyer
MyLawyer is a SaaS Based Lawyer Appointment System. In this system, a client can free register and login and make an appointment with his/her desired lawyer. Client can search lawyer based on location or chember departmentwise and appoint quickly. Lawyer can also login and create prescription for his clients, manage schedule etc.
